# `danielgiljam.com`

My website.

## Local development

Pre-requisites: [Node.js](https://nodejs.org) and [PNPM](https://pnpm.io).

1. Clone repo

2. Install dependencies

   ```sh
   pnpm install
   ```

3. Run local dev server

   ```sh
   pnpm dev
   ```

4. Open in web browser ([http://localhost:3000](http://localhost:3000) by default)
